# Step to run
## Install Rust
[here](https://www.rust-lang.org/tools/install)

## Build project
```
cargo build 
```
## Run in terminal
You can either use the [default RPC provider](https://freerpc.merkle.io/)
```
cargo run
```
or use your favorite one
```
cargo run -- --rpc-url <RPC_LINK>
```
